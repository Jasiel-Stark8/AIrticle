participant User
participant Frontend
participant Backend
participant EmailService
participant Database
participant OpenAIAPI

User -> Frontend: Access Homepage
note right of Frontend: Load assets and initialize
Frontend -> User: Display SignUp/Login options
User -> Frontend: Choose SignUp
Frontend -> Backend: InitiateSignUp(username, email, password)
note over Backend: Validate SignUp Info
alt Validation Success
    Backend -> EmailService: SendVerificationEmail(email)
    EmailService -> User: Verification Email
    User -> EmailService: ClickVerificationLink()
    EmailService -> Backend: ConfirmVerification(email)
    Backend -> Database: StoreUserData(username, email, passwordHash)
else Validation Failure
    Backend -> Frontend: ShowErrorMessage()
end
